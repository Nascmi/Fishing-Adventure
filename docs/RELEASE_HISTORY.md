# Release History

Public versions are cut deliberately. Major development checkpoints may be recorded as internal milestones without changing the package or claiming an app-store release.

## Phase 3 Complete — Five Waters & Cabin Life — 2026-07-20

- Completed the five-location journey from Backyard Pond through Open Gulf.
- Added location-specific gear, charters, day phases, keepsakes, sharing, and Passing Rain.
- Added specimen preservation, the Starter Cabin, earned Angler's Lodge, location art, and cabin collections.
- Added the earned-coin Trading Post with three permanent coin-purchased cabins.
- Reached save schema version 17 while preserving migration from the initial prototype.

This is an internal development milestone, not a public store release. Its detailed changes are archived in `CHANGELOG.md`.
