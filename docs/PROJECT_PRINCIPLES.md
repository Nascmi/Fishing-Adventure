# Project Principles

Quick reference:

- Protect the fishing loop.
- Polish before expanding.
- Respect the player's time and attention.
- Prefer calm, clear, offline-friendly experiences.
- Keep systems simple, readable, and configurable.
- Preserve saves and working behavior.
- Finish and verify what you start.
